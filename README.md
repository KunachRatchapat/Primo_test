
1. Clone repository:

   ```bash
   git clone https://github.com/KunachRatchapat/Primo_test.git
   cd Primo_test
   ```

2. Install the required dependencies:
   ```bash
    npm install
   ```
3. Build the TypeScript project:

   ```bash
    npm run build
   ```

4. Run Unit Tests:
   ```bash
   npm test
   ```
